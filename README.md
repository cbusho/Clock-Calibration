Clock-Calibration
=================

Library to set the clock speed of the MSP430

Functions
=========

void clockSet1MHZ()
  sets clock speed to 1MHZ

void clockSet8MHZ()
  sets clock speed to 8MHZ

void clockSet12MHZ()
  sets clock speed to 12MHZ

void clockSet16MHZ()
  sets clock speed to 16 MHZ
